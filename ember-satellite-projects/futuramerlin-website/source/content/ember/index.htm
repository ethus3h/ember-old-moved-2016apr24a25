Ember

Summary

Ember is an incomplete project consisting of four basic components, and several ancillary components. The components of the main project are:

  • A specification of principles and methods on which the project is based (completion: begun)
  • A community and a non-profit organization to manage and implement the project (completion: not started)
  • A computing environment (completion: begun)
  • A library of knowledge and great cultural works (completion: moderate)

This set of Web pages is the specification mentioned above (including specifications for the other three components of the project).

Ancillary components

These are secondary projects that are related to the development of the main project. The main project is not yet functioning in a manner such that it can be used, but some ancillary projects are stable and usable.

  • Crystallise, a file archival app: needs polishing
  • Warcdealer, a Web archive uploader app: advanced
  • Wreathe, a GNU/Linux distribution: moderate
  • Rubberfs, a cloud-based append-only filesystem: not started
  • Spiritual framework: moderate
  • Multimedia artwork: begun

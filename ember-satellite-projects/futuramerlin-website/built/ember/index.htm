Core project

(work-in-progress; see below for ready-to-use things)

Ember is an incomplete project consisting of four basic components, and several ancillary components. The components of the main project are:

  • A specification of principles and methods on which the project is based: begun
  • A community and a non-profit organization to manage and implement the project: not started
  • A computing environment: begun
  • The Ember Heritage Library, a library of knowledge and great cultural works: partly complete

This set of Web pages is the specification mentioned above (including specifications for the other three components of the project).

Ready-to-use tools and resources

These are secondary projects that are related to the development of the main project. The main project is not yet functioning in a manner such that it can be used, but these ancillary projects are generally usable.

  • Crystallise, a file archival app: needs polishing
  • Warcdealer, a Web archive uploader app: mostly complete

Work-in-progress ancillary projects

These are secondary projects that are related to the development of the main project, and are not yet generally usable.

  • Wreathe, a GNU/Linux distribution: partly complete
  • RubberFS, a cloud-based append-only filesystem: not started
  • Spiritual framework: partly complete
  • Multimedia artwork: begun

Overview

An app to easily archive files stored in computer-accessible digital formats.

Documentation

Usage

Crystallise provides the following user-facing command-line scripts. See the individual pages for the various scripts for usage instructions.

  • crystallise
  • decrystallise
  • quickliquid
  • dequicklify
  • crystallise-update

In addition, it also provides several scripts that it uses internally.

Installation

Requirements

  • Currently only tested on OS X.
  • An account at the non-profit Internet Archive.
  • ia (https://pypi.python.org/pypi/internetarchive) 1.0.2 or later (must be configured by running ia configure before Crystallise can be used)
  • a root login or administrator/sudo privileges for your local computer, both for installation and use of all tools except quickliquid and dequicklify.
  • wget 1.14 or later
  • hashdeep
  • (A recent version of bash)

Setup

Create a file named config.txt in your home directory, containing:

  • a title: This should be a hexadecimal UUID; to get one, visit this Web page (you want the thing at the top of the page that looks like "2F16D21E-F312-11E5-B5A1-A52D57EF8172").
  • a collection identifier: This must be a collection at the Internet Archive to which you have write access; if you don't know what to put here, put "opensource".
  • a passphrase: This must be a valid GPG passphrase.
  • the path of a directory to contain temporary files: This directory should exist, be writeable, and have enough free space for three times the amount of information being crystallised at any one time.

Each of these items should be on a separate line. They should not contain newlines.
